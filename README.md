# todo-list-app :
A simple To-Do List Web Application built using HTML, CSS, and JavaScript. Allows users to add and delete daily tasks.

# Features :
- Add Task
- Delete Task
- User Friendly UI
- Responsive Design(Mobile Friendly)
- DOM Manipulation using JavaScript

# Tech Stack Used :
- HTML
- CSS
- JavaScript
