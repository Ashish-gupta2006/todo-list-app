# todo-list-app :
This is a simple and responsive  Todo List Web Application built using HTML , CSS and JavaScript. 

# Features :
- Add Task
- Delete Task
- User Friendly UI
- Responsive Design(Mobile Friendly)
- DOM Manipulation using JavaScript

- CSS
- JavaScript
